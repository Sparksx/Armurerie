Fonctionnement :
Instanciation de la classe
	param : Array : Fichier de configuration
# L'instanciation est faite automatiquement


Public fonction load() :
Chargement du où des personnage(s)/compte(s)
À placer n'importe ou entre l'importation du plugin et l'affichage

Le paramètre 2 est facultatif et est booléen. S'il n'est pas précisé ou est à l'état faux, le premier paramètre concerne les personnages, si il est précisé et a l'état vrai, le premier paramètre concerne les comptes

	Paramètre 1 :
		int : Charge le perso id indiqué, Si aucun id ne correspond, rien n'est chargé et aucune erreur n'est retourné.
		Int  Array : chargement de tout les perso id présent dans l'array
		Aucun arg : aucune action

	return : void


Public fonction show() :
Affiche l'armurerie pour tout les perso(s)/compte(s) chargé.
Si l'un des personnage/compte entré en paramètre n'est pas déjà chargé, il est alors chargé.

Le paramètre 2 est facultatif et est booléen. S'il n'est pas précisé ou est à l'état faux, le premier paramètre concerne les personnages, si il est précisé et a l'état vrai, le premier paramètre concerne les comptes

	Paramètre 1 :
		Prend en charge les paramètres de la fonction load()
